Starter structure

The starter structure is a basic structure for holding files and folders that will function as URLs for the user.

1. Create folders and files .
├── assets
│   ├── img
home.html
index.html
page1.html

2. Insert FTML code into index.html <!doctype html > <html lang="en"> <head> <title>My application</title> <meta name="viewport" content="width=device-width, initial-scale=1, shrink-to-fit=no"> <meta http-equiv="X-UA-Compatible" content="ie=edge;ie=10;chrome=1"> <meta http-equiv="Content-Type" content="text/html; charset=UTF-8"> <base env="local:/;prod:/"> <link type="text/css" href="https://cdn.front.nu/dist/front.css" rel="stylesheet"> <link type="text/css" href="https://cdn.front.nu/dist/design/front/theme/front.css" rel="stylesheet"> <script src="https://cdn.front.nu/dist/front.js" lib="nav"></script> </head> <body> <header></header> <nav></nav> <aside></aside> <main include="home.html"></main> <aside></aside> <footer></footer> </body> </html>

3. Insert FTML code into home.html <script src="https://cdn.front.nu/dist/front.js" template="index"></script> <template> <aside index="0"></aside> <aside index="1"></aside> </template> <p>Content</p>

4. Insert FTML code into page1.html <script src="https://cdn.front.nu/dist/front.js" template="index"></script> <title>Page 1</title> <p>Content</p>
