1. Install Git on your machine.

  • macOS
  • Windows
  • Linux/Unix

2. Clone project git clone https://github.com/aleptra/front.git 3. Create an application make app:create

Application structure

1. Create folders and files .
├── assets
│   ├── img
home.html
index.html
page1.html

2. Save FTML code as index.html <!doctype html> <html lang="en"> <head> <title>My application</title> <meta name="viewport" content="width=device-width,initial-scale=1,shrink-to-fit=no"> <meta http-equiv="X-UA-Compatible" content="ie=edge;ie=10;chrome=1"> <meta http-equiv="Content-Type" content="text/html;charset=UTF-8"> <base env="local:/;prod:/"> <link type="text/css" href="https://cdn.front.nu/dist/front.css" rel="stylesheet"> <link type="text/css" href="https://cdn.front.nu/dist/design/front/theme/front.css" rel="stylesheet"> <script src="https://cdn.front.nu/dist/front.js" lib="nav"></script> </head> <body> <header></header> <nav></nav> <aside></aside> <main include="home.html"></main> <aside></aside> <footer></footer> </body> </html>

3. Save FTML code as home.html <script src="https://cdn.front.nu/dist/front.js" template="index"></script> <template> <aside index="0"></aside> <aside index="1"></aside> </template> <p>Content</p>

4. Save FTML code as page1.html <script src="https://cdn.front.nu/dist/front.js" template="index"></script> <title>Page 1</title> <p>Content</p>
